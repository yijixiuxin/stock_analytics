# stock_analytics
### 股票分析系统

> 个人学习项目
> 学习内容（Python、TuShare、Mongodb、Vue.js、Echarts）
> 通过TuShare，获取股票信息，并保存到Mongodb数据库中；
> 使用 Tornado web 框架提供股票RestFul接口服务；
> 使用 Vue.js 做为前端展示层，进行股票展示，Echarts画行情图表;
